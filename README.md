# 🎯 Optimizer VTX

Auxílio profissional AIM Assist para Free Fire.

## 🔒 Acesso

Senha: **user9090**

## ⚙️ Características

- 🎯 No recoil AIM
- 💀 Headtracker AIM
- 📊 Controle de porcentagem (100%, 90%, 80%, 70%)
- 🎮 Dois modos de mira:
  - Ao Atirar
  - Ao Olhar boneco
- 🔄 Slider de ajuste fino
- 🎮 Suporte para FF e FF MAX

## 📦 Deploy

```bash
# Instalar dependências
npm install

# Instalar Netlify CLI
npm install netlify-cli -g

# Login na Netlify
netlify login

# Inicializar projeto
netlify init

# Deploy para produção
netlify deploy --prod
```

## 🛠️ Desenvolvimento

```bash
# Rodar localmente
netlify dev
```
